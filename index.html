本テーマについて

このテーマでは、HTML,CSS,JavaScriptの基本を学び、これらを組み合わせてWebページを構成する方法を学ぶ。目標としては、Webページの構成、見た目、JSでのインタラクティブな処理についての基礎を完璧にする。

プロジェクト課題について

プロジェクト課題では、NHKの番組表、グルメ情報、世界の天気の検索の3つの中から好きなものを選び検索サイトを作る。私は、グルメ情報を選んだ。理由としては、やっぱり作るなら3つの中で一番使ったことのあるものだと思い選んだ。グルメでは、htmlで作った検索キーに行きたいお店のジャンルを書き、検索ボタンを押すとそれにあった検索結果が出るというサイトです。 project.html

数当てゲーム

kazuate.html

数当てゲームでは、1~10までの整数を3回のうちに当てるゲームです。3回のうちに答えを当てた場合は、「正解です。おめでとう！」と表示させる。3回のうちに正解できないと、「答えは、〇〇でした。すでにゲームは終わっています」と表示させる。 1回目と2回目で当たらない場合は、「まちがい、答えはもっと大きいですよ」「まちがい、答えはもっと小さいですよ」を表示させる。

各授業回の学習事項

第1回　WWW と HTML

今回の授業では、インンターネットを支える基本的な技術のWWW（World Wide Web）の仕組みと、Webコンテンツを作るHTMLの基礎を学ぶ。 具体的には、Webページの表示を支える技術要素の理解から始まり、HTML文書のヘッダーやボディや、タグを用いた情報の意味づけを習得する。さらに、要素リファレンスを活用した実践的なコーディング演習や、意図通りに表示されない原因を特定するデバッグ手法まで、Web制作の第一歩となる網羅的な知識を身につける内容となっています。 Webの背後にある仕組みを理論的に把握した上で、実際に手を動かして文書を作成し、修正するまでのプロセスを包括的に学習できるのが特徴。

演習の成果: 演習1-4:HTMLのさまざまな要素

第2回　CSS による Web ページの装飾

今回の授業では、CSSの基本構造や記述法、優先順位の仕組みを学び、Webページの見た目の装飾を習得する。 演習や開発者ツールを用いたデバッグを通じ、文字や余白の制御、効率的な外部参照の手法を実践的に理解する。
